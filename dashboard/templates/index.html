Tools

Getting Started

✓
Connect your OANDA account
Click your account name in the top-left header. Enter your OANDA Account ID and API key. The agent validates against OANDA before switching.
2
Run your first analysis in Test Mode
Enable Test Mode below. It bypasses peak-hour restrictions and uses paper money — safe to run anytime. Click ▶ Run Analysis in the header to trigger your first cycle across all pairs.
3
Understand the signal pipeline
Python calculates a confluence score for each pair using RSI, ADX, EMA crossovers, and Bollinger Bands across 3 timeframes. Only pairs scoring above the threshold are sent to Claude. Claude then adds fundamental and sentiment analysis before deciding BUY / SELL / NO_TRADE.
4
Review open positions and trade history
Open positions appear in the Live Positions table with direction, units, and a close button. Closed trades log to Trade History with entry price, SL, TP, and realized P&L.
5
Tune Test Mode parameters
The yellow banner opens the parameter panel. Adjust confluence threshold, ADX sensitivity, risk per trade, and R:R ratio. Lower thresholds = more trades. Use Increase Risk preset to stress-test the pipeline.
6
Enable auto-cycling
Once you're comfortable, enable the Auto-Cycle Scheduler above. It fires automatically at your chosen interval. Enable Sync P&L regularly to reconcile trades closed by Stop Loss or Take Profit while you weren't watching.
7
Go live
Turn off Test Mode. The agent will now execute real orders during London/NY overlap (13:00–17:00 UTC, Mon–Fri). Use Pause Agent to halt execution without stopping analysis. Close All is your emergency exit.
